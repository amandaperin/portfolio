AI Engineering

Churn Guard AI

ChurnGuard AI is an end-to-end machine learning system for customer churn prediction and business insights. It combines XGBoost modeling, statistical validation (hypothesis testing), and a FastAPI backend with an LLM-powered chatbot for explainable results. The system follows a medallion data pipeline (Bronze → Silver → Features → Gold) and is fully containerized with Docker, simulating a production environment. It identifies high-risk customers, explains key churn drivers, and supports data-driven retention strategies.

Data Engineering

Strava data warehouse

This end-to-end data engineering project automates the entire information lifecycle, from incremental extraction via the Strava API to the structuring of a dimensional model in PostgreSQL. Using a medallion architecture orchestrated by Apache Airflow, the pipeline cleans raw records and enriches them with Open-Meteo weather data, transforming physical activities into scalable, analytics-ready tables for Business Intelligence.
